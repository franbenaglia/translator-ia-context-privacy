Privacy Policy EN

Translator IA Context is developed by Francisco Arce Benaglia (Argentina). For privacy questions, contact: francisco.arcebenaglia@gmail.com.

1) What the app does

Translator IA Context helps travelers translate text from photos (e.g., restaurant menus and street signs) and speech. Translations may use AI and optional context such as device location and user preferences (e.g., dietary restrictions).

2) Data we process

  • Camera images (temporary): When you take a photo for translation, the image is used to extract/translate text. Photos are not saved on your device and not stored on our servers.
  • Audio (optional, temporary): If you use voice translation, microphone input is used to transcribe/translate speech. Audio is processed for the requested translation and is not stored by the app.
  • Location (optional): If enabled by you, precise location (GPS) may be used as context to improve translation relevance (e.g., local language usage). Location is not required to use the app.
  • User preferences: Preferences you provide (e.g., vegetarian, allergies) are used to tailor translations. By default, these preferences are stored locally on your device.
  • Text inputs: Text you type or questions you ask for translation are processed to generate the translation.

3) Third-party services and AI processing

To provide translations, the app may send the minimum necessary content (such as extracted text, text queries, and optional context like language settings and approximate context) to external processing services (e.g., AI/translation services) and/or to a backend hosted on AWS for processing.

We do not upload or store original photos. If a third-party service is used, we send only what is needed to fulfill your request.

4) How we use data

  • To translate text from images and typed input
  • To translate speech to text and provide translated output
  • To optionally generate voice output (text-to-speech)
  • To improve relevance using optional context (location and preferences)
  • To secure and operate the service

5) Data retention

  • Photos: not stored (processed temporarily and discarded).
  • Microphone audio: not stored (processed temporarily and discarded).
  • Location: not stored permanently; used only when enabled.
  • Preferences: stored locally on device unless future versions add an optional account sync feature.

6) Sharing

We do not sell personal data. Data may be shared only with service providers necessary to provide translations and operate the app (e.g., cloud hosting on AWS and external AI/translation providers), solely for app functionality.

7) Children

The app is not specifically directed to children under 13. If a parent/guardian believes a child provided personal information without consent, please contact us.

8) Your choices

  • You can use the app without enabling location (GPS).
  • You can disable microphone access and use text translation only.
  • You can clear local preferences by uninstalling the app (or via in-app reset if available).

9) Changes to this policy

We may update this policy from time to time. Updates will be posted on this page with a revised “Last updated” date.

Política de Privacidad ES

Translator IA Context es desarrollada por Francisco Arce Benaglia (Argentina). Consultas de privacidad: francisco.arcebenaglia@gmail.com.

1) Qué hace la aplicación

Translator IA Context ayuda a viajeros a traducir texto desde fotos (por ejemplo, menús y carteles) y desde voz. Las traducciones pueden apoyarse en IA y en contexto opcional como ubicación y preferencias del usuario (por ejemplo, restricciones alimentarias).

2) Datos que tratamos

  • Imágenes de cámara (temporales): al tomar una foto para traducir, la imagen se usa para extraer/traducir texto. Las fotos no se guardan en el dispositivo ni se almacenan en nuestros servidores.
  • Audio (opcional, temporal): si usas traducción por voz, el micrófono se usa para transcribir/traducir. El audio se procesa para la solicitud y no se almacena.
  • Ubicación (opcional): si el usuario lo habilita, la ubicación precisa (GPS) puede utilizarse como contexto para mejorar la relevancia. No es obligatoria para usar la app.
  • Preferencias del usuario: preferencias que brindes (p. ej. vegetariano, alergias) se usan para personalizar traducciones. Por defecto se almacenan localmente en el dispositivo.
  • Textos ingresados: texto que escribas o preguntas que hagas para traducir se procesan para generar la traducción.

3) Servicios de terceros y procesamiento con IA

Para brindar traducciones, la app puede enviar el mínimo necesario (como texto extraído, consultas de texto y contexto opcional) a servicios externos de procesamiento (por ejemplo, proveedores de IA/traducción) y/o a un backend alojado en AWS.

No subimos ni almacenamos fotos originales. Si se usa un proveedor externo, se envía únicamente lo necesario para resolver la solicitud.

4) Para qué usamos los datos

  • Traducir texto desde imágenes y texto escrito
  • Traducir voz a texto y generar salida traducida
  • Generar audio de la traducción (texto a voz), si se habilita
  • Mejorar relevancia usando contexto opcional (ubicación y preferencias)
  • Operar el servicio y mantener seguridad

5) Conservación de datos

  • Fotos: no se almacenan (se procesan temporalmente y se descartan).
  • Audio: no se almacena (se procesa temporalmente y se descarta).
  • Ubicación: no se almacena de forma permanente; se usa solo cuando el usuario la habilita.
  • Preferencias: se guardan localmente salvo que futuras versiones incorporen sincronización opcional con cuenta.

6) Compartición

No vendemos datos personales. Podemos compartir datos únicamente con proveedores necesarios para brindar la funcionalidad de la app (por ejemplo, hosting en AWS y proveedores externos de IA/traducción), solo para funcionamiento de la aplicación.

7) Menores de edad

La app no está dirigida específicamente a menores de 13 años. Si un padre/madre o tutor considera que un menor proporcionó información sin consentimiento, puede contactarnos.

8) Opciones del usuario

  • Se puede usar la app sin habilitar ubicación (GPS).
  • Se puede deshabilitar el micrófono y usar solo traducción por texto.
  • Se pueden borrar preferencias locales desinstalando la app (o mediante reset dentro de la app si existe).

9) Cambios en esta política

Podemos actualizar esta política ocasionalmente. Los cambios se publicarán en esta página con una nueva “Última actualización”.
